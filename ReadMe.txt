Language used: Python
Libraries used: Pandas,cv2

Compile steps:
- Install Python 3.6
- Set environment variable ('Path')
- Open command prompt
- Go to directory containing .py file
- Run the following command

	python ImageSegmentation.py <image file name with format> <Number of clusters>

	Sample command for random selection method:
	
		python ImageSegmentation.py "image1.jpg" 8

Resulted file will be saved as <filename>_result.jpg in the same folder
